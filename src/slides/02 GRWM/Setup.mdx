# Setup

## Instructies

- Open VS Code
- Open de terminal in VS Code <ShortcutButton text="ctrl + `" keys={['Control', '`']} />
- Ga naar de directory waar je een project wil maken
- Run command: <code className="cmd">npm create vite@latest</code>
- Volg de instructies in de terminal
- Als het create script in de terminal klaar is
  - Open de project-directory in VSCode
  - Open de terminal in VSCode <ShortcutButton text="ctrl + `" keys={['Control', '`']} />
  - Installeer de dependencies, run command: <code className="cmd">npm install</code>
  - Controleer of het gemaakte project werkt, run command: <code className="cmd">npm run dev</code>
