# Starter project downloaden

Bevat de volgende onderdelen:

- Project met waar we de vorige keer gebleven waren, zodat iedereen hetzelfde startpunt heeft
- Prettier voor code formatting
- Iedereen typescript

1. Download of clone het project:
   1. **Optie A - Download ZIP:**
      1. Ga naar https://github.com/Timdobbel/react-workshop
      1. Klik op <em>Code</em> en kies <em>Download ZIP</em>
      1. Pak het ZIP-bestand uit op een plek naar keuze
   1. **Optie B - Clone met git:**<br/>
      <code className="cmd mt-2">git clone https://github.com/Timdobbel/react-workshop.git</code>
1. Navigeer naar de projectmap: <code className="cmd">cd react-workshop</code>
1. Installeer de dependencies: <code className="cmd">npm install</code>
1. Start de development server: <code className="cmd">npm run dev</code>
1. Open je browser en ga naar `http://localhost:5173`
1. Controleer of "format on save" werkt
