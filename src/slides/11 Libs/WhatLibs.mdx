# Component Library

In frontend-development wordt veel gebruik gemaakt van libraries. React zelf is ook een library, en
Pega's `@pega/react-sdk-components` is ook een library.

Die libraries gebruiken dan bijna altijd ook weer andere libraries. Pega's SDK components maken
bijvoorbeeld gebruik van <a href="https://mui.com/material-ui/api/list/" target="_blank" rel="noreferrer">MUI</a>, een van de populairste componenten libraries.

Componenten libraries maken het makkelijker om (React) apps te maken die er goed uit zien, en die
een consistente visuele stijl hebben.

Om vooral de Pega developers daar al een klein beetje op voor te bereiden gaan we hier nu ook een
paar van die componenten gebruiken.
