# Wat is React?

## Als je een React-app bouwt, dan...

<UIfn />

Die <span className="text-cyan-300 text-xl">`f`</span> is onze React-applicatie letterlijk een functie.

Je schrijft een functie die bepaalt hoe data (state) wordt omgezet naar UI.

## React is declaratief, niet imperatief

Dat betekent: je beschrijft **wat** je wilt zien, niet **hoe** React dat moet doen.

Dus in plaats van `document.createElement('div')`, gewoon `<div>`.

_Kleine kanttekening: Uiteindelijk komt er ook nog wel wat imperatieve code bij te kijken._
