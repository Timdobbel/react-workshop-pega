# Over de workshop

1. Deze workshop is primair opgezet als voorbereiding op de Pega + React workshop.
2. <em>Maar</em> deze workshop zelf dekt puur en alleen React.

## Wat jullie straks kunnen en weten

- Begrijpen wat React is
- Zelf eenvoudige React-componenten kunnen maken en gebruiken.
- Meer complexe React-componenten kunnen lezen.
- Werken met de basic hooks.
- Lijsten en condities in JSX kunnen toepassen.
- Data kunnen laden in een React-component.
- Kennismaken met het gebruik van een component library (MUI).
- Een basis hebben om op door te bouwen.
