# Opdracht 1: Override Text Field

In deze opdracht ga je een bestaand tekstveld aanpassen (overriden) in de Pega React omgeving.

## Doel

- Leer hoe je een bestaand component kunt aanpassen naar eigen wensen.

## Stappen

1. Zoek het tekstveld dat je wilt aanpassen.
2. Override het component in je eigen code.
3. Pas bijvoorbeeld de placeholder of styling aan.

> Tip: Gebruik de documentatie van Pega React voor voorbeelden van overrides.
