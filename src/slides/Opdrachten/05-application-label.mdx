# Opdracht 5: Applicatielabel

Toon het label van de huidige applicatie in je interface.

## Stappen

1. Gebruik `PCore.getEnvironmentInfo().getApplicationLabel()` om het label op te halen.
2. Toon het label op een logische plek in je applicatie.

> Zo weet de gebruiker altijd in welke applicatie hij werkt.
