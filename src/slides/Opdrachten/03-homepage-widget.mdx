# Opdracht 3: Widget op Homepage

Maak een widget die zichtbaar is op de homepage van de applicatie.

## Stappen

1. Gebruik AppStudio om een widget toe te voegen.
2. Stel een titel en bericht in via configuratie.
3. Toon het bericht in een Alert in de widget.
4. Voeg een property toe waarmee je de severity kunt instellen: `info`, `warning`, `error`, of `success`.

> Experimenteer met verschillende severities en berichten.
